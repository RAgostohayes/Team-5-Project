How to compile a java program.

Please read instructions first before attempting to compile and run. 
1. Locate and take note of the folder where the .java file is located.
2. Open command prompt.
3. In command prompt type c: and the directory/file path to the java file you would like to complie.
4. Type "javac menu.java" <enter> 
5. If the file compiles correctly the command line will update. If it does not compile please contact the student for scolding
6. type "java menu" <enter>  Note: Do not add .java or .class to the end of the command.
